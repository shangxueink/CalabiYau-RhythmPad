使用说明

  • 点击上方"点击开始"按钮初始化音频引擎并进入3D场景。
  • 在3D场景中，移动鼠标可环顾四周。
  • 使用 W, A, S, D 键移动。
  • 屏幕中央的准心对准打击垫时，点击鼠标左键可触发对应的钢琴音。
  • 每个打击垫的播放起始时间已固定为5毫秒。
  • 每个打击垫的播放频率限制为每秒最多4次。
  • 按 `Esc` 键或长按任意键可以刷新页面并退出。
  • 按 `F11` 键可以进入/退出全屏模式。
